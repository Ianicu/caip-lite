# 产品概要设计 - 会话助手

## 1. 产品定位
**产品名称**：会话助手
**Slogan**：你的全场景智能录音专家
**核心价值**：连接4G智能录音设备与云端AI，提供从“远程录音控制”到“深度内容分析”的一站式音频知识管理解决方案。

## 2. 用户画像
- **商务人士**：需要记录会议纪要，快速回顾重点。
- **记者/访谈者**：需要远程控制设备，整理长篇采访稿。
- **学生/研究员**：记录讲座和灵感，通过AI辅助整理知识点。

## 3. 核心功能模块

### 3.1 智能设备管理 (Device Hub)
- **状态监控**：实时查看设备在线状态（4G/Wi-Fi）、剩余电量、可用存储空间、当前工作状态（空闲/录音中/上传中）。
- **远程控制**：通过App远程指令控制设备开始录音、暂停录音、结束录音并上传。
- **设备维护**：设备绑定与解绑、固件升级提醒。

### 3.2 音频资产管理 (Audio Manager)
- **云端同步**：4G网络自动上传，App端实时同步列表。
- **基础编辑**：音频裁剪、重命名、移动文件夹。
- **文件组织**：采用“合订本”（文件夹）目录结构管理音频，支持多级目录或扁平化分类。
- **录音日历**：以日历视图展示历史录音记录，方便按时间回溯。

### 3.3 AI转写与智能分析 (AI Intelligence)
- **高精度转写**：语音转文字，支持多说话人分离（Speaker Diarization）。
- **智能摘要**：基于大模型自动生成全文摘要、关键词提取、待办事项（Action Items）。
- **模板化总结**：提供多种总结模板（如：会议纪要、课堂笔记、访谈结构、灵感备忘）。
- **AI 对话 (Chat Mode)**：
    - **单条音频对话**：针对当前录音内容进行提问（如“客户提到的预算是多少？”）。
    - **合订本对话**：针对整个文件夹内的多条音频进行综合分析（如“总结这周5次晨会提到的共性问题”）。

### 3.4 个人中心 (User Center)
- **额度管理**：查看剩余转写时长/额度，充值或购买套餐。
- **账号安全**：多端登录管理。

## 4. 交互设计原则 (遵循 UI 设计风格)
- **极简高效**：首页即展示核心设备状态与最新录音，减少层级。
- **沉浸体验**：播放与转写页面采用沉浸式设计，重点突出内容。
- **科技质感**：使用 Royal Blue 与 Slate 配色，结合微交互反馈设备状态变化。
